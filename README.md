# Purple Tee LLC - Landing Page

A clean, responsive landing page for Purple Tee LLC and the Local Links golf tee-time discovery app.

## 🚀 Deployment to GitHub Pages

### Quick Setup

1. **Create a new GitHub repository** (e.g., `purpletee-landing`)

2. **Upload these files** to your repository:
   - `index.html`
   - `styles.css`
   - `privacy-policy.html`
   - `terms-of-service.html`
   - `assets/logo.png` (replace placeholder with your actual logo)
   - `README.md`

3. **Add your logo**:
   - Replace `/assets/logo.png` with your actual Local Links logo image

4. **Enable GitHub Pages**:
   - Go to repository Settings → Pages
   - Under "Source", select "Deploy from a branch"
   - Select branch: `main` (or `master`)
   - Select folder: `/ (root)`
   - Click Save

5. **Access your site**:
   - Your site will be live at: `https://[your-username].github.io/[repo-name]/`

### File Structure

```
/
├── index.html              # Main landing page
├── styles.css              # All styles (shared across pages)
├── privacy-policy.html     # Privacy policy page
├── terms-of-service.html   # Terms of service page
├── assets/
│   └── logo.png           # Local Links logo
└── README.md              # This file
```

## 🎨 Customization

### Update Logo
Replace `/assets/logo.png` with your logo file (keep the filename)

### Update Colors
Edit `styles.css` - main green color is `#166534`

### Update Content
- Company info: Edit text in `index.html`
- Legal text: Update `privacy-policy.html` and `terms-of-service.html`
- Contact email: `support@purpletee.ai` (search and replace if changing)
